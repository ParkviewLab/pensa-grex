# TaskForkStack

An Electron desktop app. Scaffold in progress — no `docs/northstar.md` yet, so
the project's intent isn't written down anywhere but this line.

## Run from source

```bash
npm ci
npm run dev      # electron-vite dev server with HMR
npm run build    # bundle to out/
npm start        # preview the built app
```
